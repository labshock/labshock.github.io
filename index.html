Network Swiftness - OT IDS

Network Swiftness is an advanced Intrusion Detection System (IDS) designed specifically for OT/ICS environments. It enhances industrial network security through real-time traffic analysis and machine learning-driven detection.

Key Features

  • 🔸 Web-Based UI: simple interface for quick traffic analysis
  • 🔸 Machine Learning Detection: no need for static rule updates
  • 🔸 OT-Specific IDS: built for industrial networks, reducing false positives
  • 🔸 OT SIEM Integration: seamless event correlation for enhanced visibility
  • 🔸 Traffic Logging: save attack traffic to continuously improve detection

Tidal Collector - OT SIEM Data Pipeline

Tidal Collector is an optimized OT SIEM data pipeline that efficiently collects, normalizes and forwards industrial security logs, ensuring better performance and cost savings.

Key Features

  • 🔸 Cost Optimization: aggregates and normalizes logs to reduce SIEM storage costs
  • 🔸 OT SIEM Integration: collects and forwards logs with minimal performance
  • 🔸 Noise Reduction: pre-processes data before sending to SIEM, improving detection efficiency
  • 🔸 Real-Time Visibility: web UI provides quick access to connected devices and log status

Pentest Fury - OT Security Testing

Pentest Fury is an advanced OT/ICS security testing tool designed to simulate real-world industrial attacks, identify vulnerabilities and enhance risk assessment.

Key Features

  • 🔸 Built for OT Security Testing: simulate real-world ICS cyberattacks
  • 🔸 Integrated with Kali Linux: leverage existing security tools in Labshock
  • 🔸 Future-Ready: Pentest Fury will introduce advanced attack simulations
  • 🔸 Targeted Assessments: designed for industrial environments to improve security posture

SCADA - Hands-On Training

SCADA Hands-On Training provides practical experience with industrial control systems, focusing on logging, visualization and real-time data handling.

Key Learning Areas

  • 🔸 SCADA Logging & Security: identify critical logs for monitoring and threat detection
  • 🔸 SCADA Fundamentals: understand how these systems control industrial processes
  • 🔸 Custom SCADA Visualization: design layouts and build interactive dashboards
  • 🔸 Data Mapping & Tags: learn how tags and data points connect to industrial devices
  • 🔸 PLC Integration: connect to programmable logic controllers for real-time data exchange

PLC - Hands-On Training

PLC Hands-On Training provides in-depth learning on programmable logic controllers, covering security, automation and industrial integration.

Key Learning Areas

  • 🔸 PLC Log Analysis: identify critical security and operational data
  • 🔸 I/O Mapping & Addressing: understand PLC memory, inputs and outputs
  • 🔸 PLC Fundamentals: learn how PLCs automate industrial processes
  • 🔸 Programming Languages: write ladder logic, function blocks and structured text
  • 🔸 SCADA Integration: connect PLCs to SCADA for real-world automation simulation

ICS Tools - Engineering Workstation

Engineering Workstation (EWS) is a pre-configured OT/ICS environment with essential security and automation tools for industrial control systems.

Key Capabilities

  • 🔸 Pre-Configured EWS: ready-to-use setup for PLC programming and SCADA deployment
  • 🔸 Remote Access Security: set up and test secure remote connections for ICS environments
  • 🔸 Industrial Protocol Analysis: monitor and analyze Modbus, DNP3 and other ICS protocols
  • 🔸 File Transfers: implement safe methods to prevent malware and unauthorized access
  • 🔸 Firewall Testing: learn how to implement secure OT network segmentation
